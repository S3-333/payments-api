# Política de seguridad

## Reportar una vulnerabilidad

Si descubres un problema de seguridad, **no** abras un issue público con los detalles.

Utiliza la pestaña **Security** del repositorio en GitHub (**Report a vulnerability**) o contacta a los mantenedores por un canal privado.

Incluye pasos para reproducir el problema y el impacto estimado cuando sea posible.
